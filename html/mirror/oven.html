NAME

oven -- Oven parameter editor

USAGE

oven

PARAMETERS

noven = 0
The oven number of the shared memory segment which should be attached.

ncomp = 0
The computer number of the shared memory segment which should be attached.

readonly = yes
If readonly is yes, do not attach the shared memory segment with write permission. Use if you want to use the parameter editor for "just looking" at the parameters and data. Non-owners of the shared memory segment must use this option. If the environment variable "USER" is not "pilot", this parameter will be silently set to yes.

remove = no
Remove the shared memory segment when you exit the parameter editor.

period = 60
The period in seconds with which the parameter editor screen is auto refreshed.

offset = 8
The number of seconds to wait after the period has expired.

DESCRIPTION

The parameter editor displays data from the shared memory segment, and allows parameters in that segment to be changed and sent to the oven V computers. A backup copy of the parameter database will be kept in the logical directory, backup, if defined in the environment.

EXAMPLES

cl> oven
