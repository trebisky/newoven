NAME

ovend -- Oven data gatherer and logger

USAGE

ovend

PARAMETERS

noven = 0
The oven number of the shared memory segment which should be attached.

ncomp = 0
The computer number of the shared memory segment which should be attached.

logdata = yes
If logdata is no, just place the data into the shared memory segment and do not write the data into the IRAF data files onto disk.

period = 60
The period in seconds with which the data is polled.

offset = 5
The number of seconds to wait after the period has expired.

DESCRIPTION

Poll the V computer every period and place the oven data into the shared memory segment. If enabled, write the data into the IRAF data files on disk.

EXAMPLES

cl> ovend &
