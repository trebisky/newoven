NAME

ovenp -- Oven parameter daemon

USAGE

ovenp

PARAMETERS

noven = 0
The oven number of the shared memory segment which should be attached.

ncomp = 0
The computer number of the shared memory segment which should be attached.

DESCRIPTION

The noven and ncomp parameters are ignored. This task attaches all the oven shared memory segments that exist when the task starts up. This is for all ovens and all V computers. The task listens for requests on a well-known TCP port and transmits the contents of the appropriate shared memory parameter database to the requester.

EXAMPLES

cl> ovenp &
