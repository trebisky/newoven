NAME

ovenr -- Oven rotation speed reader.

USAGE

ovenr

PARAMETERS

connection = oven0v0,5107
The host and port to connect to.

lower = 0
The lower speed limit.

upper = 0
The upper speed limit.

DESCRIPTION

Connect to the oven V computer and print on STDOUT the rotation speed data. Beep on STDERR if the speed is not between the limits.

EXAMPLES

cl> ovenr connection="oven0v0,5107" > rspeed.oven0v0 &
