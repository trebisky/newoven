NAME

tcmark -- print thermocouple locations on stdout

USAGE

tcmark

PARAMETERS

noven = 0
The oven number of the shared memory segment which should be attached.

ncomp = 0
The computer number of the shared memory segment which should be attached.

aspect = base
The desired subset of thermocouples. This may be "base", "lid", "wall", "alum", or "mold".

DESCRIPTION

The locations of the desired set of thermocouples are printed.

EXAMPLES

cl> tcmark
